Download Only What You Need

XEarthLayer streams satellite imagery just-in-time as you fly. Only the scenery you see gets downloaded.

Mutltiple Ortho Providers

Choose from Apple, ArcGIS, Bing, Google, MapBox, or USGS. Most are free to use.

Optimized Initial Loading

Intelligent tile prefetching and cache warming get you flying sooner.

Smart Caching

Two-tier cache system stores scenery in memory to reduce stuttering. Storage cache saves downloading scenery multiple times.

Predictive Prefetching

Uses X-Plane telemetry to prefetch tiles ahead of your aircraft. Smoother experience with fewer pop-ins during flight.

Real-Time Dashboard

Live dashboard with cache stats, download metrics, GPS position, and pipeline health. Know exactly what's happening.

Package Management

Install, update, and remove regional scenery packages with simple commands. Built-in publisher tools for content creators.
